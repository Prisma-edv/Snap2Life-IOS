    Snap2LifeIR_IOS1.0 Reference

    Snap2Life IR is the API for Image recognition provided by Prisma.

    The main class you should use is S2LIRRequestMaker: this class is a Singleton. Snap2Life IR uses AFNetworking and KissXML external libraries

    AFNetworking: https://github.com/AFNetworking/AFNetworking KissXML: https://github.com/robbiehanson/KissXML
